함수

변수에는 데이터만 저장할 수 있고, 코드는 저장할 수 없습니다. 하지만 함수를 사용하면 코드를 메모리에 저장했다가 필요할때 마다 호출하여 사용할 수 있습니다.

선언적 함수

기본형

function함수명(){
    자바스크립트 코드;
}

함수를 사용하여 코드를 저장한 것을 '함수 정의문'이라고 합니다. 변수를 선언할 때 var키워드를 사용한 것처럼 함수에서는 function 키워드를 사용해 변수를 선언합니다.

* 예제

<script>
var count = 0;

myFnc();

function myFnc(){
    count++;
    document.write("hello" +count, "<br>")
}

myFnc();

var theFnc = function(){
    count++;
    document.write("bye" +count, "<br>")
}

theFnc();
// myFnc()가 두번 실행되어 hello1hello2가 출력됩니다.
</script>

익명 함수

기본형

function함수명(){
    자바스크립트 코드;
}

함수를 리터럴 방식으로 만들고 함수의 이름이 없는 함수를 익명함수라고 합니다. 리터럴이란 함수를 문자그대로 변수에 담으면 함수 리터럴 방식으로 만드는 것입니다. 보통 함수를 재사용하지 않을 경우 익명함수를 만들어서 사용합니다. 또한 익명함수는 호이스팅이 되지 않습니다(오류발생)

* 예제

<script>
var theFnc = function(){
    count++;
    document.write("bye" +count, "<br>")
}

theFnc();
// theFnc가 한번 실행되어 bye1이 출력됩니다.
</script>

매개변수함수

기본형
function 함수명(매개변수1, 매개변수2,..... 매개변수n){
    자바스크립트 코드;
}
함수명(데이터1, 데이터2,..... 데이터n);

함수를 호출할 때 전달하고자 하는 값을 입력하여 호출할 수 있습니다. 이렇게 전달된 값은 매개변수가 받아 함수 정의문에서 사용할 수 있게됩니다.

* 예제

<script>
function myFnc(name, area){
    document.write("안녕하세요." +name+ "입니다.", "<br>");
    document.write("사는 곳은" +area+ "입니다.", "<br><br>")
}
myFnc("홍당무", "서울");

myFnc("깍두기", "부산");

결과
안녕하세요 홍당무입니다.
사는 곳은 서울입니다.
안녕하세요 깍두기입니다.
사는 곳은 부산입니다.
</script>
