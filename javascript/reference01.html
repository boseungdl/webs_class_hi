자바스크립트 선언문

선언문은 자바스크립트 코드를 작성하기 위해 필요합니다. 선언문의 시작은<script>라는 태그로 시작해서 </script>라는 태그로 끝이납니다. 

기본형 :   <script>
            자바스크립트 코드
            </script>

자바스크립트 주석처리

자바 스크립트에서 주석처리시 2가지 방법이 있습니다. 

한줄일 경우 : // 내용
                
여러 줄일 경우 : /* 
                      내용 
                     */

코드 입력시 주의 사항

1. 자바스크립트는 대, 소 문자를 구분하여 작성합니다.
2. 코드 한 줄을 작성한 후에는 세미콜론(;)을 쓰는 것이 좋습니다. 세미콜론을 쓰지 않으면 다음 예제처럼 한 줄에 2개의 코드를 작성할 경우 오류가 발생합니다.
3. 하지만 코드를 작성할 때는 한 줄에 한 문장만 작성하는 것이 가독성을 위해 좋습니다.
4. 문자형 데이터를 작성할 때는 큰 따옴표("")와 작은 따옴표('')의 겹침 오류를 주의해야 합니다.
