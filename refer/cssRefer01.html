CSS란?

css는 Style sheet 언어 입니다. HTML 문서에 있는 요소들에 선택적으로 스타일을 적용할 수 있습니다.

CSS 주석 표시

한줄일 경우 : // 내용

여러 줄일 경우 : /* 
                        내용 
                        */

(자바스크립트와 동일합니다)

CSS 선언 방법

1. 내부 스타일

internal Style : <head>태그 안에서<style>요소들을 정의

<head>
<style type="text/css">
body {
    margin: 0;
    padding: 0;
}
</style>
</head>

2. 외부 스타일

External Style : 외부 CSS파일을 링크해서 사용

<link rel="stylesheet" type="text/css" href="style.css" />

3. 인라인 스타일

inline Style : HTML 태그 안에서 style 속성을 설정

<div style="padding: 10px; border: 1px solid #000;">
    <p>contents</p>
</div>
